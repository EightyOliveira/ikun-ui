# 快速上手

## 安装
```bash
npm install ikui
```

## 引入
```js
import { createApp } from 'vue'
import App from './app.vue'

import ikui from 'ik-ui'
import 'ikui/lib/index.css'

createApp(App)
  .use(ikui)
  .mount('#app')

// or

import { IkButton } from 'ik-ui'
app.use(IkButton)

```

## 使用

```
<ik-button>按 钮</ik-button>
```
## 效果
<ik-button>按 钮</ik-button>
